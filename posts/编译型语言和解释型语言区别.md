title: 编译型语言和解释型语言区别
date: 2026-03-19 22:38:04
description: 笔记
tags:
 - 面试题
---



| **特性**     | **编译型**                  | **解释型**               |
| ------------ | --------------------------- | ------------------------ |
| **翻译时机** | 运行前（一次性）            | 运行时（逐行）           |
| **产物**     | 独立的可执行文件（如 .exe） | 无独立产物，需解释器配合 |
| **运行效率** | **极高**                    | 相对较低                 |
| **跨平台性** | 依赖特定操作系统            | 只要有解释器就能跑       |
| **典型代表** | C++, Go                     | Python, JS               |
